# AI Generation & Audio-Video Synchronization Architecture
**Project:** HackMyApartment Reel Pipeline  
**Core Challenge:** Reconciling generative Text-to-Speech variable timing with exact Remotion sequence framing.

---

## 1. The Synchronization Challenge 
Video rendering interfaces (like Remotion) structurally demand exact millisecond bounds for frame components. To feel cinematic and "viral," if an AI narration says, "Next, check out this vase," the visual overlay of the vase must transition synchronously as the character begins pronouncing the noun.

## 2. The Primary Pipeline: Word-Level Alignment API
When a user accesses the timeline generator using a fully licensed Premium API Key, the `live-preview.tsx` payload dynamically hits ElevenLabs' Advanced tracking API: `/v1/text-to-speech/{voice_id}/with-timestamps`.

1. **Extraction:** The ElevenLabs node generates the MP3 buffer alongside an `alignment` mapped timeline object detailing exact micro-second character positions.
2. **Translation:** The internal `calculateSegmentTimings()` service (found in `app/src/lib/audio-alignment.ts`) ingests these characters, identifying structural grammar bounds (such as sentences or pauses), and divides the textual structure perfectly matching the quantity of generated visual items.
3. **Execution:** It produces an `AudioTimingMapping` array dictating the EXACT `start`, `end`, and `duration` of every individual product.

## 3. The 402 Fallback: Mathematical Timeline Simulation
Due to restrictive policies established in early 2024, ElevenLabs often strictly bans standard or global IDs from hitting their external API on Free Tier workspaces, instantly rejecting them with `402 - Library Voices Restricted`.

To natively heal the broken connection and ensure user videos still build locally, the pipeline uses **Algebraic Simulation**:

1. **The Downgrade Override:** `live-preview.tsx` traps the `402` payload mid-crash, severing the network line and downgrading the payload specifically onto the Standard Free Tier endpoint (`/v1/text-to-speech`).
2. **Naked Buffer Bypass:** The Standard API flawlessly generates the Audio blob, completely sidestepping the advanced restrictions, but strips all timestamp alignment data.
3. **Structural Algebra:** Rather than crashing Remotion, the engine pushes a `null` timestamp payload natively into `calculateSegmentTimings(null, itemCount)`. The calculator algebraically divides the raw length of the audio track, distributing equally spaced time blocks across the exact numerical count of visual assets, fabricating artificial timing offsets that identically trick the compiler.

## 4. Visual Rendering Context (`video-preview.tsx`)
Regardless of whether the timelines were organically derived generated by AI or mathematically forged via the Standard endpoint downgrade, the `AudioTimingMapping` interface enforces consistency down to the video generator.

The visual layout engine utilizes `<Sequence>` mapping to structurally assemble the video components purely based on these mathematical `durationInFrames` vectors. Thus, the video editor never has to process whether the user has a Free or Paid API token; the interface simply constructs the file.
