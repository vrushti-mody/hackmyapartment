# Human vs Automation Split

## Automated daily
- Generate episode brief from theme/budget
- Prepare product candidate sheet + links + images
- Draft script + captions + CTA variants
- Build reel draft (visual + overlays + audio)
- Build story cards + community post draft

## Human daily
- Approve final products and links
- Approve final script and tone
- Choose visual variant
- Final QA on reel/story/community package
- Publish to Instagram + story + community

## Quality checklist before publish
- Links open correctly
- Price/budget claims accurate
- Visuals match products enough
- Audio timing/caption timing is clean
- CTA clear and compliant
